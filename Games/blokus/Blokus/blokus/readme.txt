○はじめに
　このソフトはIrrlichtの研究を兼ねて作成しました。
　以下のソフト・ライブラリを使用して作成しています。
　Microsoft Visual C++ 6.0
　Irrlicht 1.3.1
　IrrKlang 0.9

○動作環境
　WindowsXP Home Editionで確認しています。
　Windows98以降であれば大丈夫かと思います。

○インストール・アンインストール
　何かしら作業をする必要はありません。
　レジストリは操作していないので、フォルダごと移動/削除してください。
