Detailed Description

This should be left for internal use by irrBullet.

If you set the boolean parameter for irrBulletWorld::debugDrawWorld() to false, then you must remember to call these two lines: IVideoDriver::setMaterial(material); IVideoDriver::setTransform(irr::video::ETS_WORLD, irr::core::matrix4()); before calling irrBulletWorld::debugDrawWorld();
