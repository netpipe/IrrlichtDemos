Constructor.
The last 2 parameters will only works if there is a static object.
Please also note that due to internal conversions, if you place the moving body to a 0.0f position, the object will be moved toward a NEAR 0 value.

Parameters:
bodyAThe first body
bodyBThe second body
pivotInAThe constraint position in A (0,0,0 by default)
pivotInBThe constraint position in B (0,0,0 by default)
autoadaptIf one body is static, and this flag is setted to false. The Slide will be only orthogonal
rotatepistonIf setted to true, the dynamic object (piston) will be rotated in the slide direction.
